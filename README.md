## Weak points

- Mixed colours affects the brand identity
- Everything visual should get equal attention from user 
- Elements lacked a strict underlying grid, making the layout feel disjointed


## UX issues

- "Pending Reviews" action was visually beneath the bright attendance stats
- Too many competing visual elements at once forced the user’s brain to work harder


## Improvement

- Strategic colors like Primary Light Blue for actionable items and active states. This instantly directs the user's eye to what matters
- Grouped related data by using law of similarity
- Strict font sizing
- Unified list items (Highlights, Upcoming, Activity) into identical structural patterns so the user can scroll and scan


## Why it is better

- High-contrast text (dark gray/black on pure white) ensures the app is readable outdoors on a sunny field
- The design now uses repeatable UI tokens making it much faster for developers to build and expand the app
- Reduced decision fatigue by using progressive disclosure (hiding deep data until a card is tapped) and clear focal points
